Azure Masterclass Syllabus

Complete roadmap for the 2026 Live Certification Batch.

Module 1: Cloud Concepts & Architecture

Understand IaaS, PaaS, and SaaS. We will cover the core architectural components of Azure including Regions, Availability Zones, and Resource Groups.

Module 2: Core Services & Networking

Hands-on labs creating Virtual Machines, configuring Azure Virtual Networks (VNet), peering, and understanding load balancers.

Module 3: Security & Identity

Deep dive into Microsoft Entra ID (formerly Azure AD). Learn Role-Based Access Control (RBAC), Multi-Factor Authentication, and Zero Trust security models.
