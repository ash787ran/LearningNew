Weeks 1-3 • Foundation

Identity & Security Perimeter

We start by securing the borders. Learn how enterprise organizations manage millions of users securely using modern authentication protocols and role-based access.

Hands-on Labs Included

  • Deploy and configure Microsoft Entra ID (Azure AD).
  • Implement Conditional Access Policies & MFA.

Weeks 4-6 • Infrastructure

Global Network Topology

Master the underlying plumbing of the cloud. You will design Hub-and-Spoke networks, implement global load balancing, and connect on-premise data centers to Azure.

Hands-on Labs Included

  • Deploy Hub-and-Spoke VNet peering architecture.
  • Configure Site-to-Site VPNs & ExpressRoute mockups.
  • Global traffic routing with Azure Front Door.
VNet Peering Azure Firewall Traffic Manager
Weeks 7-9 • Modern Compute

Containers & Serverless Architectures

Move away from legacy virtual machines. Learn to architect scalable microservices that can automatically handle millions of requests while minimizing cost.

Hands-on Labs Included

  • Containerize an application using Docker & Azure Container Registry.
  • Deploy a cluster on Azure Kubernetes Service (AKS).
  • Build event-driven automation using Azure Functions.
Docker Kubernetes (AKS) Serverless
Weeks 10-12 • Mastery

Infrastructure as Code (IaC) & CI/CD

Stop configuring servers manually. You will learn to write enterprise-grade code to automatically deploy, update, and destroy entire environments in minutes.

Hands-on Labs Included

  • Write declarative infrastructure templates using Bicep.
  • State management and deployment using Terraform.
  • Automate deployments using GitHub Actions pipelines.
